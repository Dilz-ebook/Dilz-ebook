## Cara Pakai Repo Ini

### 1. Bikin Ebook PDF
1. Buka folder `ebook/`
2. Copy isi tiap file `.md` в†’ paste ke Canva (template ebook 1080x1080 atau A4)
3. Pakai font: **Heading** Cormorant Garamond / Playfair Display, **Body** Inter / Poppins
4. Warna brand: Hitam `#0A0A0A` + Emas `#C9A961` (sesuai cover referensi)
5. Export jadi PDF
6. Upload ke Lynk.id

### 2. Setup Lynk.id
1. Daftar di [lynk.id](https://lynk.id)
2. Buat produk baru, harga **Rp 25.000**
3. Copy deskripsi dari `marketing/lynkid-deskripsi.md`
4. Upload PDF ebook
5. Aktifkan auto-delivery

### 3. Mulai Promosi
1. Hari 1-3: posting konten teaser dari `marketing/thread-teaser.md`
2. Hari 4-7: posting caption promosi dari `marketing/caption-promosi.md`
3. Setiap hari: 1 hook viral dari `marketing/hook-viral-promo.md` di Threads/IG/X
4. Pin postingan launching di profil @pinokioarab
